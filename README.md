# Global Energy Transition Investment Analysis

This project analyses global energy-transition investment patterns using International Energy Agency (IEA) datasets and Python.

The analysis explores how capital is allocated between clean energy and fossil fuels across advanced and developing economies, with a focus on investment prioritisation, energy dependency, and transition dynamics.

## Key Insights
- Identified significant disparities in clean-energy investment allocation between advanced and developing economies
- Evaluated the persistence of fossil-fuel dependency across regions
- Highlighted structural differences in energy-transition financing priorities

## Objectives
- Analyse global energy-transition capital allocation trends
- Compare clean-energy and fossil-fuel investment patterns
- Evaluate differences between advanced and developing economies
- Generate policy-relevant insights through data visualisation and quantitative analysis

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- NumPy

## Analytical Techniques
- Data integration and transformation
- Schema harmonisation
- Tidy-data restructuring
- Comparative visualisation
- Percentage-share analysis
